# Personal Productivity Suite (CLI Python App)

## 📌 Overview
The Personal Productivity Suite is a command-line based Python application designed to improve daily productivity by providing multiple tools in a single interface.

The project demonstrates Python programming concepts such as:
- Object Oriented Programming
- File Handling (JSON, TXT, CSV)
- Modular Architecture
- Error Handling
- CLI Interface Design

## 🚀 Features
- Calculator (basic + scientific operations)
- Notes Manager (CRUD operations)
- Timer & Stopwatch
- File Organizer
- Unit Converter
- Backup & Restore System
- Export Notes (TXT & CSV)
